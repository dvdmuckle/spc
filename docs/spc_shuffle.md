## spc shuffle

Toggle Spotify shuffle

### Synopsis

Will toggle shuffle on or off depending on its previous state

```
spc shuffle [flags]
```

### Options

```
  -h, --help   help for shuffle
  -s, --set    Set the shuffle state to on regardless of current state
```
